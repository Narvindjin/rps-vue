Небольшая игра в камень-ножницы-бумагу, сделана на основе задачи https://www.frontendmentor.io/challenges/rock-paper-scissors-game-pTgwgvgH на Vue, включает в себя:

Ссылка: https://narvindjin.github.io/rps-vue/

- Игру с компьютером в камень-ножницы-бумагу;
- Счет сохраняется между сессиями;
- Анимации элементов в переходах между этапами и в ожидании;

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
